# Macro Tracker 🇦🇺

A calorie & macro tracking app built for Australian foods and products.

## Features
- 📋 60+ Australian foods in local database
- 🔍 Online product search (Open Food Facts AU)
- 📊 Barcode scanner for packaged products
- 📸 AI meal photo recognition (Gemini 2.5 Flash - free)
- 🕐 Recently eaten quick-add
- 📈 Fibre + full macro tracking (P/C/F/Fi)
- 📊 Weekly trends & averages
- 💾 Data saves locally in your browser

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → Sign up with GitHub
3. Click **"Add New Project"** → **"Import Git Repository"**
4. Select your repo → Vercel auto-detects Vite
5. Click **Deploy** — done!

## Setup Gemini API Key (for AI photo feature)

1. Go to [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
2. Sign in with Google → Create API Key
3. In the app, tap ⚙️ → paste your key → Save

Free tier: 250 requests/day, no credit card needed.
